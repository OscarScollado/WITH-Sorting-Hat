## Setup, running, testing, and building the project

- To setup the project:
    - Close the repository
    - Go to the root directory
    - Do `npm i`

- To run the project:
    - Do `npm run dev`
    
- To test the project:
    - Do `npm run test`

- To build the project:
    - Do `npm run build`
